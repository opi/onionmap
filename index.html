Options

You can filter out datas based on Onionoo parameters.

Examples

  • Exit nodes: ?flag=exit
  • Relays from Spain: ?country=ES
  • Relays from the Tor servers family: ?family=7BB160A8F54BD74F3DA5F2CE701E8772B841859D
  • Relays on AS29169: /as=AS29169

Note that parameters can be combined

  • Exit nodes on OVH network: ?as=AS16276&flag=exit ;)

Embed as iframe

The iframe.html url shows only the map, without any other HTML content. It helps you embed a OnionMap in an iframe.

<iframe src="/iframe.html?flag=exit" height="100%" width="100%" style="border:0;"></iframe>
